# Payment Dashboard

Next.js app for the **payments hub dashboard** (KPIs, revenue, activity). Source lives in **`my-app/`**.

**First-run / onboarding** is intentionally **not** in this repo—ship it as a **separate project** (own app or marketing flow) and deep-link users into this dashboard when they are ready.

```bash
cd my-app && npm install && npm run dev
```

Open [http://localhost:3000](http://localhost:3000) → redirects to `/payment-hub/active`. See `my-app/app/payment-hub/README.md` for routes.

**Reuse this UI shell in other apps:** see [`docs/reusing-this-layout.md`](docs/reusing-this-layout.md).
